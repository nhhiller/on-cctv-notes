# on-cctv-notes

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面本身不针对任何特定网站或域名，而是作为某些观察、记录或辅助工具的静态存储与展示。所有页面均为纯前端实现，不依赖后端服务。

## 目录说明

```
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
├── README.md       # 本文件
└── LICENSE         # 许可证文件
```

- `pages/`：每个 HTML 文件作为一个独立页面，文件名即页面标识。
- `assets/`：按需存放公共资源，部分页面也可能内联资源以减少外部依赖。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可直接在浏览器中打开使用。
- 页面内容可能涉及静态信息展示、交互工具或辅助记录功能。
- 本仓库不追踪或记录任何用户数据，所有页面均不包含外部请求或追踪代码。
- 页面可能随时间更新或补充，历史版本可通过 Git 提交记录查看。

## 维护说明
